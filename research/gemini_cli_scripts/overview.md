## Summary
Gemini CLI extends its functionality through **Agent Skills**, **Sub-agents**, and **Custom Commands**. 

### 1. Agent Skills
- **Definition:** Agent skills are specialized capabilities that can be activated to provide expert guidance and specialized tools for specific tasks (e.g., `testing`, `research`, `maintenance`).
- **Capability Categories:** These skills cover areas such as unit/widget/integration testing, codebase research, architectural refactoring, and dependency management.
- **Usage:** Skills are activated via the `activate_skill` tool, which provides the agent with a set of expert instructions and relevant resources to ensure high-quality, project-aligned outcomes.
- **Prerequisites:** Some skills may require specific environment configurations or dependencies to be present in the project.
